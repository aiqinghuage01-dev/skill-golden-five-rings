# 黄金五环采集：从定位到成交的一站式梳理

## 执行回执（硬规则）

只要本轮因为 `skill` / `技能` 指名，或因为关键词命中而进入本技能，第一行先写：
`已走技能：黄金五环采集`

若当前还只是信息收集阶段，则写：
`已走技能：黄金五环采集（信息收集阶段）`

---

## 核心原则

- **低门槛优先**：先让老板说出来，再逐步补齐，不追求一步到位
- **选择题为主**：95%的提问用"选择题+示例"，不要求经验复盘
- **多答再收敛**：允许老板一次给2-3条，AI帮他排序收敛成结论
- **行业自适应**：餐饮说"产品"，财税说"服务项目"，不讲术语
- **只加不改**：写入飞书时新增版本行，不覆盖旧数据

## 访谈交互规范

- **单题模式**：一次只问1个问题，禁止一轮多问
- 每题前先报：`当前环节` + `题号`
- 用户答完后先做1句确认复述，再进入下一题
- 若用户卡住，先给弱提示（方向+关键词），不直接代答
- 用户要求"更慢/更快/更具体"时，立即调整节奏
- 对"品牌口径/价值主张"类问题，默认先收集2-3条口语表达，再由AI收敛为1句对外版本

## 行业自适应问法

- 内部统一使用"交付单元"；对外按行业说人话：
  - 实物型（门店/工厂/电商）→ "产品/套餐/项目"
  - 服务型（财税/法务/咨询/培训）→ "服务项目/方案/顾问服务"
- 每题默认附一个行业内示例句
- 若用户反馈"听不懂"，立即切回该行业最常见说法

## 0基础用户分支

- 若用户为新手（无线上实操），禁止问"你历史数据/转化率"
- 改问期望动作与优先目标，提供可选项让用户选择
- 先完成"可执行默认方案"，后续有数据再迭代

## 采集流程（共4个环节，约15-20个问题）

### 开场白

```
你好！我来帮你用「黄金五环」方法梳理一下你的业务。

这个过程大概10-15分钟，我会一个问题一个问题问你，
大部分都是选择题，你选一个或者说几句话就行。

梳理完之后，你会得到一份清晰的定位文档，
后续AI帮你写文案、做内容都会基于这份文档来。

咱们开始？
```

用户确认后，进入环1。

---

### 基本信息（3题，快速锁定行业+昵称）

**题1：生意类型**
```
【基本信息】第1题

你做的是哪种类型的生意？

1. 店铺/同城
   餐饮、美业、教培、健身、宠物、母婴、家居、零售、药店等

2. 厂/货/农
   工厂、批发、供应链、农产品、食品加工、服装加工、建材家居等

3. 服务/咨询
   法律、财税、设计、装修、保险、中介、咨询、代运营等

选1、2或3就行。
```

**题2：具体生意**
```
【基本信息】第2题

具体做的什么生意？用一句大白话说。

比如："教小孩画画的""做川菜的""修德系车的""给工厂做模具的"
```

**题3：怎么称呼你**
```
【基本信息】第3题

别人平时怎么叫你？

比如："老刘""张姐""小王""刘哥"

就说平时客户或者朋友叫你的那个称呼就行。
```

AI根据老板回答，自动推导行业、细分赛道、大致客群，不再逐个追问。

### 定位（4题）

**题3：人设关键词**
```
【定位】第3题

你希望客户觉得你是什么样的人？选2-3个：

1. 专业（技术过硬、经验丰富）
2. 实在（不忽悠、不宰人）
3. 源头（工厂直供、自己生产）
4. 有温度（关心客户、售后好）
5. 有故事（创业经历、行业沉淀）
6. 其他（你说）
```

**题4：信任背书**
```
【定位】第4题

客户凭什么相信你？说2-3条你最硬的底气：

比如：
- "做了15年了"
- "服务过3000个客户"
- "某某品牌指定供应商"
- "回头客占80%"

能想到几条说几条，说不出来也没关系，我帮你挖。
```

**题5：人设经历（可选）**
```
【定位】第5题（选答，跳过也行）

你有没有什么创业故事或者转折经历？

比如：
- 从打工到自己开店的过程
- 遇到过什么大坎是怎么过来的
- 为什么选择了这个行业

有就说几句，没有就说"跳过"。
```

### 产品（4题）

**题6：主力产品**
```
【产品】第6题

你现在最主要靠什么赚钱？（你的主力{交付单元}是什么）

比如：
- 餐厅："人均80的商务套餐"
- 美甲店："268的日式美甲套餐"
- 工厂："定制模具，客单价5万起"

说一个你卖得最多/利润最好的。
```

**题7：产品梯队**
```
【产品】第7题

除了主力{交付单元}，你有没有：

1. 低价引流款（让新客户先体验，几十块到几百块的）
2. 高端利润款（大客户/老客户升级的，几千到几万的）
3. 复购/会员制（客户定期回来的）
4. 目前只有一个产品

有的话分别说一下大概什么价位。
```

**题8：核心差异化**
```
【产品】第8题

客户为什么买你的不买隔壁的？说2-3个理由：

1. 品质更好（用料/技术/效果）
2. 价格更优（同品质更便宜）
3. 服务更好（售后/体验/态度）
4. 更专业（专注某个领域）
5. 更方便（位置/速度/流程）
6. 其他（你说）

每个理由最好能举个例子或者数据。
```

**题9：差异化证据**
```
【产品】第9题

你刚才说的这些优势，有没有硬证据？

比如：
- "用的是进口XXX材料，成本比同行高30%"
- "返修率不到1%"
- "大众点评4.9分"
- "回头客占70%"

有什么说什么，说不出来的我标记为"待补充"，后面再找。
```

### 成交（4题）

**题10：客户为什么必须买**
```
【成交】第10题

客户如果不来找你，会怎样？

1. 问题会越来越严重（比如车越拖越贵修、牙越拖越疼）
2. 会多花冤枉钱（比如找不靠谱的被坑）
3. 错过最好的时机（比如旺季前不准备就来不及了）
4. 其他（你说）
```

**题11：客户为什么现在买**
```
【成交】第11题

你平时怎么让客户"现在就来"而不是"再想想"？

1. 限时优惠（本周特价/月底截止）
2. 限量名额（每天只接X单）
3. 赠品福利（买XX送XX）
4. 季节/时效（现在不做就来不及了）
5. 目前没有这类设计

选一个或者说说你的做法。
```

**题12：转化钩子**
```
【成交】第12题

你希望看到你内容的人做什么动作？

1. 直接到店（线下门店生意）
2. 加微信咨询
3. 打电话预约
4. 在线下单/团购
5. 进直播间了解
6. 其他

选一个主要的就行。
```

**题13：客户场景**
```
【成交】第13题

客户一般在什么情况下会想到你？

比如：
- "车刮了/该保养了"
- "想换个发型/皮肤不好"
- "要办公司/报税"
- "想给孩子找个培训班"

说1-2个最常见的场景。
```

## 通用规则

### 零配置，即插即用
- 不需要 API Key，不依赖任何外部服务
- 复制整个文件夹到任何电脑，直接可用
- 行业自适应，任何行业老板都能用

### {交付单元} 替换规则
- SKILL.md 中出现的 `{交付单元}` 是占位符，AI 必须根据题1的行业类型自动替换为行业术语：
  - 实物型（门店/工厂/电商）→ 替换为"产品"或"套餐"
  - 服务型（财税/法务/咨询/培训）→ 替换为"服务项目"或"方案"
- 替换后在对话中直接使用替换后的词，不展示占位符原文

### 卖点分层规范
- 知识库禁止只保留"一句版本"，必须输出主副卖点分层：
  - 主打卖点：1条（头部优先）
  - 次级卖点：3-5条（按优先级排序）
- 每条卖点提供至少3种表达变体，避免内容同质化
- 每条卖点标注适用场景：短视频开场 / 直播转化 / 朋友圈 / 私聊承接

### 异常处理
- 老板说"不知道"或"跳过" → 标记为"待补充"，不阻断流程
- 老板答非所问 → 温和引导回当前问题，不强纠
- 中途要退出 → 保存已采集的部分，标记状态为"采集中"
